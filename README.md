# ATM Machine Mini Project

This is a beginner-friendly ATM Machine project using Python.

## Features
- Account holder name
- PIN login system
- 3 wrong PIN attempts block account
- Check balance
- Deposit money
- Withdraw money
- Change PIN with old PIN verification
- Confirm new PIN
- Block account after 3 wrong old PIN attempts

## Concepts Used
- Variables
- If-else
- While loop
- Nested loop
- Input/output
- Basic condition checking
